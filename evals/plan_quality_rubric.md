# Plan Quality Rubric

Score each category 0-2.

| Category | 0 | 1 | 2 |
|----------|---|---|---|
| Problem clarity | vague | understandable | sharp and user-specific |
| Target user | missing | broad | specific ICP/persona |
| First step | missing | broad | small, buildable, valuable |
| Evidence | none | assumptions labeled | sourced evidence or clear research queue |
| Scope control | scope creep | partial constraints | clear goals/non-goals |
| Architecture | absent | generic | fits product and risk |
| Risks | absent | listed | mitigated with gates |
| Tasks | absent | rough | buildable and gated |
| Metrics | absent | vanity | product-specific success metrics |
| Handoff | absent | vague | next agent can continue |

## Passing Bar

Minimum recommended score: 90%.
