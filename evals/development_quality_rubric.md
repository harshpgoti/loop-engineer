# Development Quality Rubric

Score each category 0-2.

| Category | 0 | 1 | 2 |
|----------|---|---|---|
| Task alignment | unrelated | partial | directly satisfies task |
| Minimal diff | broad churn | moderate | tight and focused |
| Correctness | broken | plausible | handles expected cases |
| Tests | none | partial | relevant and passing |
| Security/risk | ignored | noted | mitigated/tested |
| Docs | missing | partial | updated where useful |
| Review | none | self-summary | findings-oriented review |
| Handoff | missing | vague | precise next step |

## Passing Bar

Minimum recommended score: 90%.
